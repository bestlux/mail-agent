# Calendar Brief

- Use `list_calendars` if the target calendar is not obvious.
- Use `get_events` with a bounded time range.
- Summaries should lead with the next meetings, then conflicts, then missing prep or travel concerns.
- Do not imply calendar creation or mutation support in v1.
