# @iomancer/mail-agent-shared

Internal support package for [`mail-agent`](https://github.com/bestlux/mail-agent).

This package contains shared runtime paths, config parsing, secret-store selection, trust-policy helpers, and common types used by the Mail Agent daemon and plugin CLI.

Most users want the top-level package instead:

- [`mail-agent`](https://github.com/bestlux/mail-agent/tree/main/packages/plugin)

Use this package directly only if you are developing Mail Agent itself or embedding its shared runtime pieces.
